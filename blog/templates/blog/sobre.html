A Jornada no Senac Tech

Este projeto foi desenvolvido como parte prática das aulas da Trilha de Python no Senac Tech. Mais do que um simples site, ele representa a aplicação real dos conceitos estudados em sala de aula.

O desenvolvimento foi realizado de forma colaborativa com a turma, avançando passo a passo e seguindo os exemplos de cada módulo. A evolução do código foi contínua, transformando teorias de lógica e web design em uma aplicação funcional.

Tecnologias Utilizadas

Python Django HTML5 / CSS3 SQLite / Banco de Dados
